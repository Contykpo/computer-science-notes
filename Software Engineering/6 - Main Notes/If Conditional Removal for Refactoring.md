# If Conditional Removal

1. Create a polymorphic hierarchy with an abstraction of the if conditional.
2. Move the body of each if statement to each corresponding abstraction using the same message.
3. Name the abstractions from (1).
4. Name the message from (2).
5. Replace the if statement for the sending of the polymorphic message.
6. Search the polymorphic object (optional).
